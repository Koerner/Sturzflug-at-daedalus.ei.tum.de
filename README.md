Sturzflug-at-daedalus.ei.tum.de
===============================

Sturzflug @ daedalus

This is a students project of TUM (technische Univerität München). 

It's written to 
-locate an little air ship,
-calculate a track,
-and move it on this Track.

For more details see www.daedalus.ei.tum.de [GERMAN only]
